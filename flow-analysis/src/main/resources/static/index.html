i 口径说明（指标定义与计算口径）
流入 inflow
相邻窗口间，从其他区域进入本区域的人次（from ≠ R 且 to = R）。
流出 outflow
相邻窗口间，从本区域离开到其他区域的人次（from = R 且 to ≠ R）。
留存 retained
相邻窗口仍停留在本区域的人次（from = R 且 to = R），即 OD 矩阵的对角线。
位置口径 loc
每个用户在一个时间窗口的归属区域 = 该窗口末切片的主导区域（按时间 argMax 取最后出现的区域）。
transition-only（仅转移）
仅统计相邻两窗口都在网（有定位）的用户的区域转移；离线（无定位）的切片不产生记录，不计入流入 / 流出 / 留存。
在网人数 population
population = retained + inflow（窗口内总在网人数 = 留存 + 流入）。
时间口径
时区 Asia/Shanghai；窗口按粒度对齐（hour=整点 / day=整日）；查询区间为左闭右开 [start, end)。
